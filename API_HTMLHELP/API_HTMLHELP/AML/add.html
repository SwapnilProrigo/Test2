AML Actions

add

Add an Item as an instance of an ItemType.

Syntax:

<Item type = "Type" action= "add" [id="GUID"]>
 <property>value</property>
 ...
 <Relationships>
  <Item type="Type" action="add|create" [id="GUID"]>
   <property>value</property>
   ...
     <related_id>
      <Item type="Type" action="add|create|get" [item attributes]>
       <property>value</property>
        ...
      </Item>
     <related_id>
  </Item>
   ...
 </Relationships>
</Item>
 

Notes:

Add creates a new Item on the server of the specified type.  Relationships to related Items can also be defined using the <Relationships> tag.

Examples:

<!-- Add part with specified properties -->
<Item type="Part" action="add">
 <item_number>ABCD-1234</item_number>
 <name>Transistor</name>
</Item>

<!-- Add part with new related item and quantity specified -->
<Item type="Part"  action="add">
 <item_number>AAAA-1000</item_number>
 <name>Cable Assembly</name>
 <classification>Assembly</classification>
  <Relationships>
    <Item type="Part BOM" action="add">
      <quantity>2</quantity>
      <related_id>
        <Item type="Part" action="add">
          <item_number>1121-9011</item_number>
          <name>Binder clip</name>
        </Item>
      </related_id>
    </Item>
  </Relationships>
</Item>
