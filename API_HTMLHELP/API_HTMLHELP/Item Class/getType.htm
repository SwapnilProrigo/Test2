Item Class

getType()

Returns value of the 'type' attribute of the Item node.

C#

if (this.getType()=="Part") {
 this.setAttribute("select", "item_number, cost");
}
return this.apply();

VB:

If Me.getType() = "Part" Then
 Me.setAttribute("select", "item_number, cost")
End If
Return Me.apply()

Notes:

Returns the yalue of the item 'type' attribute if the attribute exists, empty string otherwise.

Example above check the current item type and if equal to "Part" set the select attribute of the item.
