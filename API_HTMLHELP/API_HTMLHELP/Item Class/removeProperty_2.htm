Item Class

removeProperty(String property , String language)

Removes the property tag with the specified name and language from the item dom.

C#:

if (this.getProperty("country")!= "France") {
 this.removeProperty("contact_name", "fr");
}

VB:

If Me.getProperty("country") <> "France" Then
 Me.removeProperty("contact_name", "fr")
End If

Notes:

Remove property from dom if exists.
