Item Class

isError()

Returns true if the item is an "error item".

C#

Innovator inn = this.getInnovator();
Item cal = this.newItem("Business Calendar Year", "get");
cal.setProperty("year", "2031");
Item result = cal.apply();
if (result.isError()) return inn.newResult("Calendar not found");
return result;

VB:

Dim inn As Innovator = Me.getInnovator()
Dim cal As Item = Me.newItem("Business Calendar Year", "get")
cal.setProperty("year", "2031")
Dim result As Item = cal.apply()
If result.isError() Then Return inn.newResult("Calendar not found")
Return result

Notes:

Returns true if error item.
