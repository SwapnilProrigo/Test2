Item Class

ToString()

Returns AML string that represents the current Item.

C#

Item customer = this.newItem("Customer", "get");
customer.setProperty("name", "ACME");
customer = customer.apply();
string customerAML = customer.ToString();
return this.getInnovator().newResult(customerAML);
 

VB:

Dim customer As Item = Me.newItem("Customer", "get")
customer.setProperty("name", "ACME")
customer = customer.apply()
Dim customerAML As String = customer.ToString()
Return Me.getInnovator().newResult(customerAML)
 

Notes:

Override the ToString() method to allow for item inspection, etc.
