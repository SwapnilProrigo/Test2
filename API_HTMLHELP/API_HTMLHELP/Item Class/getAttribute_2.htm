Item Class

getAttribute(String attribute String defaultValue)

Returns value of the attribute with the specified name on the item's node.  If attribute is not found then return default value.

C#:

if (this.getAttribute("select", "")=="") {
 this.setAttribute("select", "id,item_number");
}
return this;
 

VB:

If Me.getAttribute("select", "") = "" Then
 Me.setAttribute("select", "id,item_number")
End If
Return Me

Notes:

Returns attribute value or default if the attribute doesn't exist.

The examples above check to see if the current item has a select attribute.  If not, then empty string is returned from method and then attribute value is set.
