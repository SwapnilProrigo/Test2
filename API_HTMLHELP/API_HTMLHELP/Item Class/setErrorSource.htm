Item Class

setErrorSource(String source)

Sets error source on the "error" item.

C#:

Innovator inn = this.getInnovator();

Item err = inn.newError("");

err.setErrorCode("-1");
err.setErrorDetail("My custom message");
err.setErrorSource("My program");
err.setErrorString("Custom Error Message");

return err;

VB:

Dim inn As Innovator = Me.getInnovator()

Dim err As Item = inn.newError("")

err.setErrorCode("-1")
err.setErrorDetail("My custom message")
err.setErrorSource("My program")
err.setErrorString("Custom Error Message")

Return err


 

Notes:

Set the error item source.
