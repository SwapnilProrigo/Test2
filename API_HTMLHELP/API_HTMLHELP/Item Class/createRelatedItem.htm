Item Class

createRelatedItem(String type, String action)

Creates an item with the specified type and action and sets it as the related item on the current item.

C#

Item part = this.newItem("Part" , "get");
Item partRel = this.newItem("Part BOM", "get");
part.addRelationship(partRel);
partRel.createRelatedItem("Part", "get");

Item relatedItem = partRel.getRelatedItem();
relatedItem.setProperty("item_number", "8120-8900");
return part.apply();

VB:

Dim part As Item = Me.newItem("Part", "get")
Dim partRel As Item = Me.newItem("Part BOM", "get")
part.addRelationship(partRel)
partRel.createRelatedItem("Part", "get")

Dim relatedItem As Item = partRel.getRelatedItem()
relatedItem.setProperty("item_number", "8120-8900")
Return part.apply()

Notes:

If property related_id doesn't exist on this item, it's created. Returned item shares dom property with this item (i.e. both returned item's and this item's dom reference the same instance of Aras XmlDocument). The method is similar to setRelatedItem(Item) except that setRelatedItem(Item) gets related item as method argument and doesn't create it internally.

In the examples above, a list of parent Parts will be returned that have a Part BOM relationship to a Part named "8120-8900".  The related_id is created using createRelatedItem and then the item_number property is set on the related item before the request is applied to the server.
