Item Class

isLogical()

Find out if the item's node is a "logical" node ("AND", "OR", "NOT").

C#

if (logicItm.isLogical()) return this.getInnovator().newResult("Logic operators found");
return this;
 

VB:

If logicItm.isLogical() Then
 Return Me.getInnovator().newResult("Logic operators found")
End If
Return Me
 

Notes:

Returns true if item in dom contains logical   "AND" | "OR" | "NOT" .
