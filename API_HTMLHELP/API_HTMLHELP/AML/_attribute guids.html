AML Attributes

GUID List (idlist)

Every item in Aras Innovator is assigned a guaranteed unique identifier which consists of 32 alphanumeric characters. Certain actions also support a list of ids to act on multiple items in a single action.

Syntax:

idlist="B7DE10503F3A4C08BA056628921A440E,C178DF9A420C4E8A8C9A058F27D662CF, ..."

Notes:

The GUID list is assigned the to the "idlist" attribute.

Examples:

<!-- Promote specified parts-->
<Item type="Part" action="promoteItem" idlist="B7DE10503F3A4C08BA056628921A440E,C178DF9A420C4E8A8C9A058F27D662CF">
<state>In Review</state>
</Item>
