Innovator Class

applyAML(String)

Sends AML request to Innovator server and returns server response as an item.

C#:

Innovator inn = this.getInnovator();
string AML = "<AML><Item type='Part' action='get'/></AML>";
Item result = inn.applyAML(AML);

VB:

Dim inn as Innovator = Me.getInnovator
Dim AML as String = _
 "<AML><Item type='Part' action='get'/></AML>"
Dim result as Item = inn.applyAML(AML)

JavaScript:

var inn = this.getInnovator();
var AML = "<AML><Item type='Part' action='get'/></AML>";
var result = inn.applyAML(AML);

Notes:

It's assumed that passed AML script has a root tag <AML> that contains one or more <Item> children elements.

The method send the AML script to the server and returns an Item object where the dom property is the xml response returned from the server.
