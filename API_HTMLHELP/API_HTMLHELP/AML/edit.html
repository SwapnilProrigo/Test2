AML Actions

edit

Lock,  update and unlock an Item.

Syntax:

<Item type="Type" action="edit" [item attributes]>
 <property>value</property>
 ...
 <Relationships>
  <Item type="Type" action="add|create" [id="GUID"]>
   <property>value</property>
   ...
     <related_id>
      <Item type="Type" action="add|create|get" [item attributes]>
       <property>value</property>
        ...
      </Item>
     <related_id>
  </Item>
   ...
 </Relationships>
</Item>
 

Notes:

The edit action requires item attribute be either an existing GUID or a where clause to prevent an ambiguous query.  All three actions (lock, update, unlock) must be successful to commit the changes.

Examples:

<!-- Edit part with specified properties -->
<Item type="Part" action="edit" where="[part].item_number='ABCD-1234'">
 <name>Carriage Assembly</name>
 <description>Supports multi-track component installation.</description>
</Item>

<!-- Edit existing part and add existing related item and quantity specified -->
<Item type="Part" action="edit" where="[part].item_number='C8111A'">
    <Relationships>
      <Item type="Part BOM" action="add">
        <quantity>1</quantity>
        <related_id>
          <Item type="Part" action="get">
            <item_number>8120-8900</item_number>
          </Item>
        </related_id>
      </Item>
    </Relationships>
</Item>
