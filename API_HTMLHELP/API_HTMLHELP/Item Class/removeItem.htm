Innovator Class

removeItem(Item)

Remove item from current Item's nodelist.

C#:

Item itmlist =  this.newItem("Customer", "get");
itmlist.setAttribute("select", "name");
itmlist = itmlist.apply();
Item itm_find =
itmlist.getItemsByXPath("//Item[name='IBM']");
itmlist.removeItem(itm_find);  //Remove Item from collection
return itmlist;

VB:

Dim itmlist As Item = Me.newItem("Customer", "get")
itmlist.setAttribute("select", "name")
itmlist = itmlist.apply()
Dim itm_find As Item = itmlist.getItemsByXPath("//Item[name='IBM']")
itmlist.removeItem(itm_find)
'Remove Item from collection
Return itmlist

Notes:

The Item being removed must exist on the current Item's nodelist.  In these examples the Item to be removed is located with XPath.
