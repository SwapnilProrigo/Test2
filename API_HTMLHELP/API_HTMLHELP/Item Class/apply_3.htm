Item Class

apply(String action)

Applies the AML Item request on the server.  The action parameter determines how the request will be processed.

C#:

Item itm =  this.newItem("Customer");
itm.setProperty("name", "ACME");
itm.setProperty("contact_name", "Andrew Harris");
return itm.apply("add");
 

VB:

Dim itm As Item = Me.newItem("Customer")
itm.setProperty("name", "ACME")
itm.setProperty("contact_name", "Andrew Harris")
Return itm.apply("add")

Notes:

Action parameter overwrites current action set on the Item request (if any).

Returns an Item where the dom property is the XML returned from the server.

The dom may contain an Item, a collection of Items, or an error.
