Item Class

isRoot()

Checks if this.node is the root item in this.dom

C#:

Item parent = relatedItem.getParentItem();
if( parent.isRoot()) {
parent.setProperty("cost", total);
}

VB:

Dim parent As Item = relatedItem.getParentItem()
If parent.isRoot() Then
parent.setProperty("cost", total)
End If


 

Notes:

Returns boolean true if item is root item in dom.
