Innovator Class

getNewID()

Return a new 32 character hex string based on a GUID.

C#:

Innovator inn = this.getInnovator();
string newid = inn.getNewID();
Item addItem = this.newItem("Customer", "add");
addItem.setID(newid);
addItem.setProperty("name", "New Customer");
return addItem.apply();

VB:

Dim inn As Innovator = Me.getInnovator()
Dim newid As String = inn.getNewID()
Dim addItem As Item = Me.newItem("Customer", "add")
addItem.setID(newid)
addItem.setProperty("name", "New Customer")
Return addItem.apply()

Notes:

ID returned is guaranteed to be unique for current Aras enterprise database.
