Item Class

setProperty(string propertyName, String value, String lang)

Set the property with the specified name using supplied language. If property doesn't exist on the item, create it first and then set.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME", "fr");
return itm.apply();
 

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME", "fr")
Return itm.apply()
 

Notes:

If property has attribute is_null set to 1 and specified property value is not an empty string, the attribute is removed from the property. If specified property value is null, then attribute is_null='1' is set on the property and it's value is set to empty string (i.e. after setting null value the property tag will look like: <prop is_null='1' />).
