Item Class

getProperty(string propertyName, string defaultValue)

Gets value of the property with the specified name.  If property not found return default value.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME");
itm = itm.apply();
string contact = itm.getProperty("contact_name", "");

if (contact != "") {
  return this.getInnovator().newResult(contact);
}
else {
 return this.getInnovator().newError("No contact name"); 
}

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME")
itm = itm.apply()
Dim contact As String = itm.getProperty("contact_name", "")
If contact <> "" Then
 Return Me.getInnovator().newResult(contact)
Else
 Return Me.getInnovator().newError("No contact name")
End If

Notes:

If the property is an item-property, ID of the item-property is returned. If the property doesn't exist or it's an item-property without ID, the default value is returned; otherwise the method returns value of the specified property.

If the property has attribute is_null set to 1 and the property value is empty string then the property value is interpreted as null.
