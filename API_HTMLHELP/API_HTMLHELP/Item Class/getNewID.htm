Item Class

getNewID()

Generate new 32 character hex string globally unique identifier.

C#:

string newID = this.getNewID();

Item customer = this.newItem("Customer", "add");
customer.setID(newID);
customer.setProperty("name", "Thomas Inc.");
customer.apply();

return this.getInnovator().newResult("Item created with: " + newID);

VB:

Dim newID As String = Me.getNewID()

Dim customer As Item = Me.newItem("Customer", "add")
customer.setID(newID)
customer.setProperty("name", "Thomas Inc.")
customer.apply()

Return Me.getInnovator().newResult("Item created with: " + newID)

Notes:

ID is guaranteed to be unique in the current Aras enterprise database.

This method is helpful when ID of new item needs to be established before Item is created on the server (use for federated foreign key, etc.).
