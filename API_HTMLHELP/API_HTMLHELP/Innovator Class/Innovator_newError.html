Innovator Class

newError(String errMsg)

Returns a new "error" item and displays error string.

C#:

Innovator inn = this.getInnovator();
string cost = this.getProperty("cost", "");
if (cost == "") {
 return inn.newError("Cost required!");
}
return this;
 

VB:

Dim inn As Innovator = Me.getInnovator()
Dim cost As String = Me.getProperty("cost", "")
If cost = "" Then
 Return inn.newError("Cost required!")
End If
Return Me

Notes:

The passed message is set as InnerText on the <faultstring> element without any pre-processing, so symbols like '<', '>', etc. will be escaped.

See the Item class setError... methods to further customize the returned error Item.
