Innovator Class

fetchDefaultPropertyValues(bool overwrite_current)
 

Fetches the default ItemType property values (from the database) and sets them on the item dom.

C#

Item customer = this.newItem("Customer", "add");
customer.fetchDefaultPropertyValues(true);
customer.setProperty("name", "Dempsey Corporation");
return customer.apply();

VB:

Dim customer As Item = Me.newItem("Customer", "add")
customer.fetchDefaultPropertyValues(True)
customer.setProperty("name", "Dempsey Corporation")
Return customer.apply()

Notes:

If 'overwrite_current is true', overwrite any existing property values on the item.

If a property doesn't have a default value defined on the ItemType, the property is not set.
