Item Class

setID(String id)

Sets the id on an item.

C#:

Item part = this.newItem("Part", "add");
this.setProperty("item_number", "8000-1010");
this.setProperty("name", "Transistor 300mv");
string newID = this.getNewID();
part.setID(newID);
return part.apply();

VB:

Dim part As Item = Me.newItem("Part", "add")
Me.setProperty("item_number", "8000-1010")
Me.setProperty("name", "Transistor 300mv")
Dim newID As String = Me.getNewID()
part.setID(newID)
Return part.apply()

Notes:

Sets the id property on item.
