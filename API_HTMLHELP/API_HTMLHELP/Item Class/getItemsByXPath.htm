Item Class

getItemsByXPath(string xpath)

Gets an item (single or collection) by resolving the xPath specified.

C#:

Item qry = this.newItem("Part", "get");
qry = qry.apply();
Item itm_find =
qry.getItemsByXPath("//Item[item_number='T-0011-2010']");
return this.getInnovator().newResult(itm_find.getID());
 

VB:

Dim qry As Item = Me.newItem("Part", "get")
qry = qry.apply()
Dim itm_find As Item = qry.getItemsByXPath("//Item[item_number='T-0011-2010']")
Return Me.getInnovator().newResult(itm_find.getID())

Notes:

The returned item shares its dom with this item, i.e. returned item dom and this.dom reference the same instance of ArasXmlDocument.
