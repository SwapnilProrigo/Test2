Innovator Class

getNextSequence(String sequence)

Returns the next sequence value by sequence name.

C#:

Innovator inn = this.getInnovator();
string nextseq = inn.getNextSequence("ECR");
 

VB:

Dim inn As Innovator = Me.getInnovator()
Dim nextseq As String = inn.getNextSequence("ECR")

Notes:

Current value of named sequence is incremented after successful method execution.
