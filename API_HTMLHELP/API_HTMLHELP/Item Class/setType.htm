Item Class

setType(String type)

Sets value of the 'type' attribute of the Item node.

C#

Item part = this.newItem();
part.setType("Part");
part.setAction("add");
this.setProperty("item_number", "8000-1010");
this.setProperty("name", "Transistor 300mv");
return part.apply();
