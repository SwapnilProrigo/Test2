Item Class

fetchFileProperty(string propertyName, string targetPath, mode)

Download a file from a Vault. Calling the method means that the physical file will be streamed to the targetPath when called.

JavaScript:

var fileItem = this.fetchFileProperty("_my_file", "C:\\TrainingFiles", FetchFileMode.Normal);

C#:

Item fileItem = this.fetchFileProperty("_my_file", "C:\\TrainingServerFiles", FetchFileMode.Normal);

Notes:

This method is only available in Aras Innovator Version 11 and greater and replaces the checkout method which is deprecated.
If used in a Client Method, file is streamed to client directory, otherwise file is streamed to server directory.
The property must be of type Item with a datasource of FIle. 
A File item is returned is successful - the checkout_path property contains the full name of the retrieved file.
FetchFileMode.Normal streams the file, FetchFileMode.Dry only returns the File item and does not stream the file.
An apply() is NOT required.
