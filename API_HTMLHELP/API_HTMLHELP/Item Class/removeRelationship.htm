Item Class

removeRelationship(Item relationshipItem)

Remove <Relationships> node from item dom.

C#:

Item partrel = this.newItem("Part Document", "get");

if (this.getType()!="Part") {
 this.removeRelationship(partrel);
}

VB:

Dim partrel As Item = Me.newItem("Part Document", "get")

If Me.getType() <> "Part" Then
Me.removeRelationship(partrel)
End If
 

Notes:

Remove <Relationships> node from request.
