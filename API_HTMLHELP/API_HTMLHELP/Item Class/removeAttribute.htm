Innovator Class

removeAttribute(String attributeName)

Removes the specified attribute from the dom item if the attribute exist; does nothing otherwise.

C#:

Innovator inn = this.getInnovator();
Item countResult = inn.applySQL("select count(*) as count from innovator.part");
int count = Convert.ToInt32(countResult.getProperty("count"));

Item vendor = this.newItem("Part", "get");
vendor.setAttribute("maxRecords", "100");

if (count < 1000) vendor.removeAttribute("maxRecords");

return  vendor.apply();

VB:

Dim inn As Innovator = Me.getInnovator()
Dim countResult As Item = inn.applySQL("select count(*) as count from innovator.part")
Dim count As Integer = Convert.ToInt32(countResult.getProperty("count"))

Dim vendor As Item = Me.newItem("Part", "get")
vendor.setAttribute("maxRecords", "100")

If count < 1000 Then
 vendor.removeAttribute("maxRecords")
End If

Return vendor.apply()

Notes:

Remove item attribute if it exists.

In the examples above the maxRecords attribute is removed from the request if the part count is less than 1000.
