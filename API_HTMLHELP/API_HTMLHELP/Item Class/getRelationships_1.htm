Item Class

getRelationships()

Returns an item that is a collection of ALL relationships items available on the instance.

C#:

Item part = this.newItem("Part", "get");
part.setProperty("item_number", "8120-8900");
part = part.apply();
part.fetchRelationships("Part BOM");
part.fetchRelationships("Part Document");
Item relationships = part.getRelationships();
return this.getInnovator().newResult(relationships.getItemCount().ToString());

VB:

Dim part As Item = Me.newItem("Part", "get")
part.setProperty("item_number", "8120-8900")
part = part.apply()
part.fetchRelationships("Part BOM")
part.fetchRelationships("Part Document")
Dim relationships As Item = part.getRelationships()
Return Me.getInnovator().newResult(relationships.getItemCount().ToString())

Notes:

Returns an Item that is a collection of all relationships items available on the instance. If this doesn't have relationships, the returned item nodeList will be empty.

The methods does NOT make a request to the server but rather returns relationships items available on this.dom in memory. In order to get relationships from the server use the fetchRelationships(...) methods.

The examples above get any relationships of type Part BOM or Part Document on a selected Part and return the total count.
