Item Class

isEmpty()

Returns true if item count is zero.

C#

Item file = this.newItem("File", "get");
file.setProperty("filename", "Surveys.xml");
file = file.apply();
if (file.isEmpty()) {
 return this.getInnovator().newError("No files found");
}
return file;

VB:

Dim file As Item = Me.newItem("File", "get")
file.setProperty("filename", "Surveys.xml")
file = file.apply()
If file.isEmpty() Then
 Return Me.getInnovator().newError("No files found")
End If
Return file
 

Notes:

Returns boolean true if item count = 0.
