Item Class

isCollection()

Checks if the instance represents a set of items and not a single item.

C#

Item file = this.newItem("File", "get");
file.setProperty("filename", "Survey.xml");
file = file.apply();
if (file.isCollection()) {
 return this.getInnovator().newError("More than one file found with that name");
}
return file;

VB:

Dim file As Item = Me.newItem("File", "get")
file.setProperty("filename", "Survey.xml")
file = file.apply()
If file.isCollection() Then
 Return Me.getInnovator().newError("More than one file found with that name")
End If
Return file
 

Notes:

Returns boolean true if item count is greater than one.
