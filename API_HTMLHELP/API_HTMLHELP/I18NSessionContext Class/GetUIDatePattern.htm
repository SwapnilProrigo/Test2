I18NSessionContext Class

GetUIDatePattern(String innovatorDatePattern)
 

Returns locale specific date-time pattern that corresponds to the passed Innovator pattern (e.g. 'short_date').

C#:

I18NSessionContext cntx = this.getInnovator().getI18NSessionContext();
string code = cntx.GetUIDatePattern("short_date_time");
return this.getInnovator().newResult(code);
 

VB:

Dim cntx As I18NSessionContext = Me.getInnovator().getI18NSessionContext()
Dim code As String = cntx.GetUIDatePattern("short_date_time")
Return Me.getInnovator().newResult(code)

Notes:

The examples above return the following if the context is set to fr-FR (French France):

<Result>dd/MM/yyyy HH:mm:ss</Result>
