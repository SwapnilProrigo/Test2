Item Class

isNew()

Checks the attribute 'isNew' on the item.

C#

if (this.isNew()) {
 return this.getInnovator().newError("Item has never been saved.");
}
return this;
 

VB:

If Me.isNew() Then
 Return Me.getInnovator().newError("Item has never been saved.")
End If
Return Me
 

Notes:

Returns true if isNew attribute of item is '1'.  New items are created by the Item.newItem(..) or Innovator.newItem(..) methods.  An item is considered new if it has not been applied to the server.
