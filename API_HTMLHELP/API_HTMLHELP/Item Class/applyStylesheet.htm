Item Class

applyStyleSheet(String xslStylesheet, String type)

Performs XSL transformation of the item's DOM and returns the result of the transformation based on xslStylesheet argument.

C#:

string ss = "file:///c:/Sample.xslt";
Item itm = this.newItem("Part", "get");
itm = itm.apply();
var ret= itm.applyStylesheet(ss,"URL");
return this.getInnovator().newResult(ret);

VB:

Dim ss As String = "file:///c:/Sample.xslt"
Dim itm As Item = Me.newItem("Part", "get")
itm = itm.apply()
Dim ret = itm.applyStylesheet(ss, "URL")
Return Me.getInnovator().newResult(ret)

JavaScript:

var ss = "file:///c:/Sample.xslt";
var itm = this.newItem("Part", "get");
itm = itm.apply();
var ret= itm.applyStylesheet(ss,"URL");
return ret;
 

Notes:

The type argument can be either 'text' or a 'url' (case insensitive).

If type='url' the xslStylesheet argument is interpreted as file URL. It means that user can pass to the method URLs like: "http://localhost/mystylesheets/stylesheet1.xslt" or "file:///c:/tmp/stylesheet2.xslt"; or absolute or relative path to the file, e.g. "c:/tmp/stylesheet2.xslt" or "../../stylesheet3.xslt".

The examples above use the stylesheet below to return back a list of Parts in an html table.

<?xml version="1.0" ?>
<xsl:stylesheet
    version = "1.0"
    xmlns:xsl = "http://www.w3.org/1999/XSL/Transform"
    xmlns:soap = "http://www.w3.org/2003/05/soap-envelope">
    <!--Output to HTML -->
    <xsl:output method = "html"/>
    <!--Match to soap envelope -->
    <xsl:template match = "soap:Envelope/soap:Body">
        <html>
            <xsl:apply-templates/>
        </html>
    </xsl:template>
    <xsl:template match = "Result">
        <html>
            <h2>Order Items</h2>
            <table border = "1">
                <tr bgcolor = "#9acd32">
                    <th>Item Number</th>
                    <th>Instructions</th>
                </tr>
                <xsl:for-each select = "Item">
                    <tr>
                        <td>
                            <xsl:value-of select = "id/@keyed_name"/>
                        </td>
                        <td>
                            <xsl:value-of select = "instructions"/>
                        </td>
                    </tr>
                </xsl:for-each>
            </table>
        </html>
    </xsl:template>
</xsl:stylesheet>
