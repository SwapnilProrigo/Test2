I18NSessionContext Class

ConvertUtcDateTimeToNeutral(String utcStr, String inPattern)
 

Converts utcStr date to string representing the same moment of time in the time zone of the session in neutral format.

C#:

string utc_date = "2013-05-02T09:00:00.000Z";

I18NSessionContext cntx = this.getInnovator().getI18NSessionContext();
string date_value = cntx.ConvertUtcDateTimeToNeutral(utc_date, "");

return this.getInnovator().newResult(date_value);

VB:

Dim utc_date As String = "2013-05-02T09:00:00.000Z"

Dim cntx As I18NSessionContext = Me.getInnovator().getI18NSessionContext()
Dim date_value As String = cntx.ConvertUtcDateTimeToNeutral(utc_date, "")

Return Me.getInnovator().newResult(date_value)

Notes:

Accepts the following parameters:

utcStr - A string that represent a date-time in the UTC pattern.
inPattern - Invariant culture date time format.

In the examples above the following is returned if the session context is fr-FR (French France);

<Result>2013-05-02T05:00:00</Result>
