Item Class

newItem(String)

Returns a new Item node with the specified type.

C#:

Innovator inn = this.getInnovator();
Item itm = inn.newItem("Customer");
itm.setAction("get");
return itm.apply();

VB:

Dim inn As Innovator = Me.getInnovator()
Dim itm As Item = inn.newItem("Customer")
itm.setAction("get")
Return itm.apply()

Notes:

Item is established with supplied ItemType name.

Use the Item method setAction to set item attribute before applying to server.
