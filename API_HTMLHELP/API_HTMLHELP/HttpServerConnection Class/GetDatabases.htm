HttpServerConnection Class

GetDatabases()

Returns list of possible databases to use with this connection.  Requires unrestricted connection.

C#:

string url = "http://localhost/InnovatorServer";
HttpServerConnection cnx = IomFactory.CreateHttpServerConnection(url);
string[] databases = cnx.GetDatabases();
return this.getInnovator().newResult(string.Join(" ", databases));

VB:

Dim url As String = "http://localhost/InnovatorServer"
Dim cnx As HttpServerConnection = IomFactory.CreateHttpServerConnection(url)
Dim databases As String() = cnx.GetDatabases()
Return Me.getInnovator().newResult(String.Join(" ", databases))

Notes:

Returns a string array of configured databases for this server connection.
