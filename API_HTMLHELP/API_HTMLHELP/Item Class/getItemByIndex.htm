Item Class

getItemByIndex(int index)
 

Gets an item by index.

C#:

Item customers = this.newItem("Customer", "get");
customers = customers.apply();

for (int i=0; i < customers.getItemCount(); i++) {
 Item itm_idx = customers.getItemByIndex(i);
 if (itm_idx.getProperty("address_state", "") != "") {
  itm_idx.setAction("edit");
  itm_idx.setProperty("country", "United States");
  itm_idx.apply();
  }
}
return customers;

VB:

Dim customers As Item = Me.newItem("Customer", "get")
customers = customers.apply()
For i As Integer = 0 To customers.getItemCount() - 1
 Dim itm_idx As Item = customers.getItemByIndex(i)
 If itm_idx.getProperty("address_state", "") <> "" Then
  itm_idx.setAction("edit")
  itm_idx.setProperty("country", "United States")
  itm_idx.apply()
 End If
Next
Return customers

Notes:

This method is used to iterate over a collection of items using a zero based index number (0 = first item in collection).

Returned item shares its dom with this item (i.e. returned item and this item dom property reference the same instance of Aras XmlDocument).   To debug the indexed item view the _node property of the item.

The examples above change the country of any Customer that have an "address_state" that is not blank.
