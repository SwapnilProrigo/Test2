Item Class

newItem()

Create a new empty Item node.

C#:

Innovator inn = this.getInnovator();
Item itm = inn.newItem();
itm.setType("Customer");
itm.setAction("get");
return itm.apply();

VB:

Dim inn As Innovator = Me.getInnovator()
Dim itm As Item = inn.newItem()
itm.setType("Customer")
itm.setAction("get")
Return itm.apply()

Notes:

Item is established without an ItemType or action.

Use the Item methods setType and setAction to set item attributes before applying to server.
