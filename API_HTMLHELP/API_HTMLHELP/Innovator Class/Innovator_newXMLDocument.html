Innovator Class

newXMLDocument()

Creates new Aras XmlDocument.

C#:

XmlDocument doc = this.getInnovator().newXMLDocument();
XmlElement root = doc.CreateElement("AML");
XmlElement item = doc.CreateElement("Item");
item.SetAttribute("type", "Part");
item.SetAttribute("action", "get");
doc.AppendChild(root);
root.AppendChild(item);
Item itm = this.newItem();
itm.loadAML(doc.OuterXml);
itm = itm.apply();
 

VB:

Dim doc As XmlDocument = Me.getInnovator().newXMLDocument()
Dim root As XmlElement = doc.CreateElement("AML")
Dim item As XmlElement = doc.CreateElement("Item")
item.SetAttribute("type", "Part")
item.SetAttribute("action", "get")
doc.AppendChild(root)
root.AppendChild(item)
Dim itm As Item = Me.newItem()
itm.loadAML(doc.OuterXml)
itm = itm.apply()

Notes:

Creates a new System.XML.XmlDocument to allow for XML processing within an Aras Method.

Examples above create a new AML message in an XMLDocument and the pass the document XML to the Aras server.
