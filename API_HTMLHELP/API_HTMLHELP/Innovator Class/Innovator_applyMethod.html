Innovator Class

applyMethod(String, String)

Apply a method by name, passing a string value as its body. Returns item passed from calling Method.

C#:

Innovator inn = this.getInnovator();
string body ="<itm_type>Part</itm_type>
    <itm_num>T-0029-2010</itm_num>";
Item res = inn.applyMethod("GetItem",body);
return res;

VB:

Dim inn as Innovator = Me.getInnovator
Dim body as String = _
 "<itm_type>Part</itm_type>" & _
 "<itm_num>T-0029-2010</itm_num>"
Dim res as Item = _
 inn.applyMethod("GetItem", body)
 Return res

JavaScript:

var inn = this.getInnovator();
var body ="<itm_type>Part</itm_type>
<itm_num>T-0029-2010</itm_num>";
var res = inn.applyMethod("GetItem",body);
return res;

Notes:

 The named Method must be a Server Method.  Any number of parameters can be passed in the body string argument. The item returned is passed back from the called Method.

The examples above call the followiing server Method:

//GetItem (Called Method)
string item_number = this.getProperty("itm_num");
string item_type = this.getProperty("itm_type");
Item myItem = this.newItem(item_type, "get");
myItem.setProperty("item_number", item_number);
myItem = myItem.apply();
return myItem
