Item Class

createPropertyItem(String propName, String type, String action)

Creates an item with the specified type and action and sets it as the specified property of this item.

C#

Item part = this.newItem("Part" , "add");
part.setProperty("name", "Cable Harness");
part.setProperty("item_number", "CH-1021");

part.createPropertyItem("owned_by_id", "Identity", "get");
Item identity = part.getPropertyItem("owned_by_id");
identity.setProperty("name", "Component Engineering");

return part.apply();

VB:

Dim part As Item = Me.newItem("Part", "add")
part.setProperty("name", "Cable Harness")
part.setProperty("item_number", "CH-1021")

part.createPropertyItem("owned_by_id", "Identity", "get")
Dim identity As Item = part.getPropertyItem("owned_by_id")
identity.setProperty("name", "Component Engineering")

Return part.apply()

Notes:

This method creates a new item (using the type and action specified) on an property of datatype ItemType.  In the examples above the owned_by_id property is assigned to the Identity named "Component Engineering".

The method is similar to setPropertyItem(propName, item) except that setPropertyItem(propName, item) gets an item already created as the method argument and doesn't create it internally.
