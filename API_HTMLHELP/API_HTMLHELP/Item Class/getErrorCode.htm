Item Class

getErrorCode()

Gets error code of the "error" item.

C#:

string faultCode="";
Item part = this.newItem("Program", "get");
part = part.apply();

if (part.isError()) {
 faultCode = part.getErrorCode();
 }

return this.getInnovator().newResult(faultCode);

VB:

Dim faultCode As String = ""
Dim part As Item = Me.newItem("Program", "get")
part = part.apply()

If part.isError() Then
 faultCode = part.getErrorCode()
End If

Return Me.getInnovator().newResult(faultCode)

Notes:

Returns value of <faultcode>. If the item is not an "error" item, empty string is returned.
