Innovator Class

getItemByKeyedName(String itemType, String keyname)

Returns an Item object that matches the ItemType name and keyed name for the Item.

C#:

Innovator inn = this.getInnovator();
string keyname = "Innovator Admin";
Item myItem = inn.getItemByKeyedName("User", keyname);
return myItem;

VB:

Dim inn As Innovator = Me.getInnovator()
Dim keyname As String = "Innovator Admin"
Dim myItem As Item = inn.getItemByKeyedName("User", keyname)
Return myItem

Notes:

If request to the server failed the method returns an "error" item; if no item with specified type and id found in the database the method returns 'null'; otherwise the method returns the found item.
