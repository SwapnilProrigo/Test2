Innovator Class

getConnection()

Returns the server connection set on the current Aras Innovator instance.

C#:

Innovator inn =  this.getInnovator();
HttpServerConnection cnx = (HttpServerConnection) inn.getConnection();
 

VB:

Dim inn As Innovator = Me.getInnovator()
Dim cnx As HttpServerConnection = DirectCast(inn.getConnection(), HttpServerConnection

Notes:

Server connection object that implements IServerConnection interface.

The real type of the server connection object depends on what was passed to the IomFactory.CreateInnovator(IServerConnection) when the instance of the Innovator class was created.
