Item Class

removeProperty(String property)

Removes the property tag with the specified name from the item dom.

C#:

if (this.getProperty("country")!= "United States") {
 this.removeProperty("zip_code");
}

VB:

If Me.getProperty("country") <> "United States" Then
 Me.removeProperty("zip_code")
End If

Notes:

Remove property from dom if exists.  Note that this does not delete the property in an item that already exists in the database.
