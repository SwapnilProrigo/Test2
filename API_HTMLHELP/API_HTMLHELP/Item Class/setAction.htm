Item Class

setAction(String action)

Set the 'action' attribute on this.node.

C#:

Item part = this.newItem("Part");
part.setAction("get");
part.setProperty("item_number", "8120-8900");
return part = part.apply();

VB:

Dim part As Item = Me.newItem("Part")
part.setAction("get")
part.setProperty("item_number", "8120-8900")
Return
 

Notes:

Set the action attribute to item.
