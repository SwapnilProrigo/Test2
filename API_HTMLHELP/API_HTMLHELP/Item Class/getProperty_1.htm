Item Class

getProperty(string propertyName)
 

Gets value of the property with the specified name.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME");
itm = itm.apply();
string contact = itm.getProperty("contact_name");
return this.getInnovator().newResult(contact);
 

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME")
itm = itm.apply()
Dim contact As String = itm.getProperty("contact_name")
return Me.getInnovator().newResult(contact)

 

Notes:

If the property is an item-property, ID of the item-property is returned. If the property doesn't exist or it's an item-property without ID, null is returned ( in JavaScript undefined is return) ; otherwise the method returns value of the specified property.
