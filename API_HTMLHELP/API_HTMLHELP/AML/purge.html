AML Actions

purge

Remove an Item version..

Syntax:

<Item type="Type" action= "purge" [item attributes]>
</Item>
 

Notes:

The purge action requires Item attribute id is an existing GUID or GUID list.

Examples:

<!-- Purge specified part version-->
<Item type="Part" action="purge" id="B7DE10503F3A4C08BA056628921A440E">
</Item>
