AML Actions

update

Update a locked Item.

Syntax:

<Item type="Type" action="update" [item attributes]>
 <property>value</property>
 ...
 <Relationships>
  <Item type="Type" action="add|create|delete|get" [item attributes]>
   <property>value</property>
   ...
     <related_id>
      <Item type="Type" action="add|create|get" [item attributes]>
       <property>value</property>
        ...
      </Item>
     <related_id>
  </Item>
   ...
 </Relationships>
</Item>
 

Notes:

Update requires that the Item specified is locked.  See the lock action.  Relationships to related Items can also be defined using the <Relationships> tag.

Examples:

<!-- Update locked part with specified property value -->
<Item type="Part" action="update" id="97FC2C2999264A1B8E9ECC36CE6CF702">
  <description>Revision posted May 2013</description>
</Item>

<!-- Update locked part with new related item and quantity specified -->
<Item type="Part"  action="update" id="97FC2C2999264A1B8E9ECC36CE6CF702">
 <name>Cable Assembly</name>
 <classification>Assembly</classification>
  <Relationships>
    <Item type="Part BOM" action="add">
      <quantity>2</quantity>
      <related_id>
        <Item type="Part" action="add">
          <item_number>1121-9011</item_number>
          <name>Binder clip</name>
        </Item>
      </related_id>
    </Item>
  </Relationships>
</Item>

<!-- Update locked part and delete specified relationship -->
<Item type="Part" action="update" id="97FC2C2999264A1B8E9ECC36CE6CF702">
 <Relationships>
  <Item type="Part BOM" action="delete" id="09GC2C2999264A1B8E9ECC36CE6CF702">
  </Item>
 </Relationships>
</Item>
