Innovator Class

applySQL(String)

Sends SQL request to Innovator server and returns server response as an Item.

C#:

Innovator inn = this.getInnovator();
Item qry = inn.applySQL("select login_name from innovator.[user]");
return qry;
 

VB:

Dim inn As Innovator = Me.getInnovator()
Dim qry As Item = _
 inn.applySQL("select login_name from innovator.[user]")
Return qry

Notes:

The method returns back a Result collection containing each row as an Item.

The applySQL method is intended for execution in server Methods only.   Attempting to execute SQL that modifies data in a client Method (by a user without administrative credentials) will result in an error.

The examples above return the following:

<Result>
<Item><login_name>admin</login_name> </Item>
<Item><login_name>root</login_name></Item>
<Item><login_name>serviceuser</login_name></Item>
<Item> <login_name>vadmin</login_name></Item>
</Result>
