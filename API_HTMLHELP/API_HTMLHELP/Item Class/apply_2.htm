Item Class

apply(Hashtable argsObject)

Applies the AML Item request on the server.  Sets properties on the Item request based on Hashtable key value pairs.

C#:

Hashtable ht = new Hashtable();
ht.Add("name", "ACME");
ht.Add("contact_name", "Andrew Harris");
Item itm =  this.newItem("Customer", "add");
return itm.apply(ht);

VB:

Dim ht As New Hashtable()
ht.Add("name", "ACME")
ht.Add("contact_name", "Andrew Harris")
Dim itm As Item = Me.newItem("Customer", "add")
Return itm.apply(ht)

Notes:

A collection of name-value pairs where 'name' is interpreted as 'property name' and 'value' as the property value. Property values are set prior to applying the AML action.

Returns an Item where the dom property is the XML returned from the server.

The dom may contain an Item, a collection of Items, or an error.
