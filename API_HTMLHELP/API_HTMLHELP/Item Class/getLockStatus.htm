Item Class

getLockStatus()

Gets an item's lock status based on the property locked_by_id.

C#:

Item customer = this.newItem("Customer", "get");
customer.setProperty("name", "IBM");
customer = customer.apply();
int lockStatus= customer.getLockStatus();

VB:

Dim customer As Item = Me.newItem("Customer", "get")
customer.setProperty("name", "IBM")
customer = customer.apply()
Dim lockStatus As Integer = customer.getLockStatus()

Notes:

Possible return values are:

  • 0 not locked

  • 1 locked by current user

  • 2 locked by other
