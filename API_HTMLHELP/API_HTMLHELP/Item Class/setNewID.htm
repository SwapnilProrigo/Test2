Item Class

setNewID()

Gets new ID by calling getNewID() then passes obtained ID to setID(id).

C#:

Item part = this.newItem("Part", "add");
this.setProperty("item_number", "8000-1010");
this.setProperty("name", "Transistor 300mv");
part.setNewID();
return part.apply();

VB:

Item part = this.newItem("Part", "add");
this.setProperty("item_number", "8000-1010");
this.setProperty("name", "Transistor 300mv");
part.setNewID();
Return part.apply();

Notes:

Sets the  id property on item to new GUID.
