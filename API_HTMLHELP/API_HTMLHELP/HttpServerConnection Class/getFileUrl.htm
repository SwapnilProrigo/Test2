HttpServerConnection Class

getFileURL(String, URLType)

Returns file full file url vault location. Can also return AuthenticationBroker's token.

C#:

HttpServerConnection cnx = IomFactory.CreateHttpServerConnection(url, db, user, pw);
Item login_result  = cnx.Login();
Item file = innovator.newItem("File", "get");
file.setProperty("filename", "customers.txt");
file = file.apply();
string fileid = file.getID();
string url =cnx.getFileUrl(fileid, UrlType.None);
return inn.newResult(url);
 

VB:

Dim cnx As HttpServerConnection = IomFactory.CreateHttpServerConnection(url, db, user, pw)
Dim login_result As Item = cnx.Login()
Dim file As Item = innovator.newItem("File", "get")
file.setProperty("filename", "customers.txt")
file = file.apply()
Dim fileid As String = file.getID()
Dim url As String = cnx.getFileUrl(fileid, UrlType.None)
Return inn.newResult(url)
 

Notes:

Possible URLType for enumeration - None or SecurityToken
