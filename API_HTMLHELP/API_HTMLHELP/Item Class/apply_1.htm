Item Class

apply()

Applies the AML Item request on the server.

C#:

Item itm =  this.newItem("Customer", "add");
itm.setProperty("name", "ACME");
itm.setProperty("contact_name", "Andrew Harris");
return itm.apply();
 

VB:

Dim itm As Item = Me.newItem("Customer", "add")
itm.setProperty("name", "ACME")
itm.setProperty("contact_name", "Andrew Harris")
Return itm.apply()

Notes:

Returns an Item where the dom property is the XML returned from the server.

The dom may contain an Item, a collection of Items, or an error.
