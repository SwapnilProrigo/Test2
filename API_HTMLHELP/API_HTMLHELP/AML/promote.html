AML Actions

promoteItem

Promote Item(s) to next lifecycle state.

Syntax:

<Item type="Type" action="promoteItem" [item attributes]>
 <state>value</state>
 </Item>
 

Notes:

Promote requires that the Item specified is unlocked and the state value is the legal next life cycle state for the item(s) being promoted. The user (identity) of the current session must be a member of the transition identity defined in the life cycle map.

Examples:

<!-- Promote single part  -->
<Item type="Part" action="promoteItem" id="97FC2C2999264A1B8E9ECC36CE6CF702">
  <state>In Review</state>
</Item>

<!-- Prmote multiple parts -->
<Item type="Part"  action="promoteItem" idlist="97FC2C2999264A1B8E9ECC36CE6CF702,C178DF9A420C4E8A8C9A058F27D662CF">
 <state>In Review</state>
</Item>
