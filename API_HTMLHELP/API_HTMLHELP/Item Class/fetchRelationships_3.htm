Item Class

fetchRelationships(String relationshipName, String selectList, String orderBy)

Fetches relationships of specified type from the server and sets them on the item.  Only returns relationship item properties included in the selectList sorted by orderBy properties.

C#:

Item part = this.newItem("Part" , "get");
part.setProperty("item_number", "8120-8900");
part = part.apply();

Item partWithRelationships = part.fetchRelationships("Part Document", "id,related_id", "created_on");
Item relationships = partWithRelationships.getRelationships("Part Document");
int count = relationships.getItemCount();

return this.getInnovator().newResult(count.ToString());

VB:

Dim part As Item = Me.newItem("Part", "get")
part.setProperty("item_number", "8120-8900")
part = part.apply()

Dim partWithRelationships As Item = part.fetchRelationships("Part Document", "id,related_id", "created_on")
Dim relationships As Item = partWithRelationships.getRelationships("Part Document")
Dim count As Integer = relationships.getItemCount()

Return Me.getInnovator().newResult(count.ToString())

Notes:

This method retrieves relationship data from the database an applies it to an existing item.  The item returned from this method includes the current item with relationship data appended (based on selectList).  To access the relationship data requires the use of the getRelationships method as shown.

This method makes a server request without performing an apply.
