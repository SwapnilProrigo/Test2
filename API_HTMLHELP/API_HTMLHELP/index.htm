Aras Innovator Training Examples

This reference help file is provided as a training supplement to the standard InnovatorAPIReference help file to show you working examples of:

  • AML built in Method Actions

  • Innovator Object Model (IOM) Class methods:

  • HttpServerConnection Class

  • I18NSessionContext Class

  • Innovator Class

  • IomFactory Class

  • Item Class

 

You can copy and paste sample code from the help file to your own programs to see how each method works.  Each topic contains a small section of sample code with the selected method or action highlighted for easy reference:

Hashtable ht = new Hashtable();
ht.Add("name", "ACME");
ht.Add("contact_name", "Andrew Harris");
Item itm = this.newItem("Customer", "add");
return itm.apply(ht);

Most IOM examples are shown in C# as well as VB.Net.  Several examples that are typically run on the client are also shown in JavaScript.
