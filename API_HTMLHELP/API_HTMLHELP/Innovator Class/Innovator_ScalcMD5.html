Innovator Class

ScalcMD5(String text)

Computes the MD5 hash value for the specified string.

C#:

string md5_string = Innovator.ScalcMD5("mypassword");

VB:

Dim md5_string = Innovator.ScalcMD5("mypassword")

Notes:

The MD5 Message-Digest Algorithm is a widely used cryptographic hash function that produces a 128-bit (16-byte) hash value.

Passwords stored in Aras Innovator use this hash calculation to store the string data.
