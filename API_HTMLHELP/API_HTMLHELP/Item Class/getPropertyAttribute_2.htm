Item Class

getPropertyAttribute(string propertyName, string attributeName, string defaultValue)

Gets the specified attribute of the property with the specified name or return defaultValue if attribute not found.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME");
itm = itm.apply();
string state = itm.getPropertyAttribute("current_state", "keyed_name", ""); 
if (state!= "") {
return this.getInnovator().newResult(state);
}
else {
return this.getInnovator().newError("No current state");
}

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME")
itm = itm.apply()
Dim state As String = itm.getPropertyAttribute("current_state", "keyed_name", "")
If state <> "" Then
 Return Me.getInnovator().newResult(state)
Else
 Return Me.getInnovator().newError("No current state")
End If

Notes:

If either property with the specified name or the specified attribute on the property doesn't exist, returns defaultValue provided.
