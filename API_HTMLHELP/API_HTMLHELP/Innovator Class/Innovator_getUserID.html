Innovator Class

getUserID()

Returns id of the user assosiated with a given instance of Innovator if this user is currently logged in, otherwise throws "Not logged in" exception.

C#:

Innovator inn = this.getInnovator();
string uid = inn.getUserID();
Item user = inn.getItemById("User", uid);
return inn.newResult(user.getProperty("keyed_name"));

VB:

Dim inn As Innovator = Me.getInnovator()
Dim uid As String = inn.getUserID()
Dim user As Item = inn.getItemById("User", uid)
Return inn.newResult(user.getProperty("keyed_name"))

Notes:

Returns current 32 character id (GUID) of user currently logged into Aras session.
