I18NSessionContext Class

GetDefaultLanguageSuffix()

Returns code of the default language suffix.

C#:

I18NSessionContext cntx = this.getInnovator().getI18NSessionContext();
string code = cntx.GetDefaultLanguageSuffix();
return this.getInnovator().newResult(code);
 

VB:

Dim cntx As I18NSessionContext = Me.getInnovator().getI18NSessionContext()
Dim code As String = cntx.GetDefaultLanguageSuffix()
Return Me.getInnovator().newResult(code)

Notes:

The examples above get the default language suffix set on the server.
