Item Class

getErrorString()

Returns the error message.

C#:

string faultMessage="";
Item part = this.newItem("Program", "get");
part = part.apply();

if (part.isError()) {
 faultMessage= part.getErrorString();
 }

return this.getInnovator().newResult(faultMessage);

VB:

Dim faultMessageString = ""
Dim part As Item = Me.newItem("Program", "get")
part = part.apply()

If part.isError() Then
 faultMessage= part.getErrorString()
End If

Return Me.getInnovator().newResult(faultMessage)

Notes:

The returned value is obtained from the <faultstring> tag of <Fault>. If the instance is not an error item an emtpy string is returned.
