I18NSessionContext Class

GetTimeZone()

Returns locale of the current session.

C#:

I18NSessionContext cntx = this.getInnovator().getI18NSessionContext();
string zone= cntx.GetTimeZone();
return this.getInnovator().newResult(zone);
 

VB:

Dim cntx As I18NSessionContext = Me.getInnovator().getI18NSessionContext()
Dim zone As String = cntx.GetTimeZone()
Return Me.getInnovator().newResult(zone)

Notes:

The examples above get the timezone for the current session.
