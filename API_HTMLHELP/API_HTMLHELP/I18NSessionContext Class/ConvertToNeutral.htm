I18NSessionContext Class

ConvertToNeutral(String svalue, String vtype, String datePtrn)

Converts value that is a string representation of value of type vtype into the neutral format. It is assumed that svalue has a format that is a valid format for the session locale.

C#:

string formatted_date = "02/05/2013";
string float_num = "41,00027357629127";
string decimal_num = "234,12";

I18NSessionContext cntx = this.getInnovator().getI18NSessionContext();
string date_value = cntx.ConvertToNeutral(formatted_date, "date", "short_date");
string float_value = cntx.ConvertToNeutral(float_num, "float", "");
string decimal_value = cntx.ConvertToNeutral(decimal_num, "decimal", "");

return this.getInnovator().newResult(date_value + " " + float_value + " " + decimal_value);

VB:

Dim formatted_date As String = "02/05/2013"
Dim float_num As String = "41,00027357629127"
Dim decimal_num As String = "234,12"

Dim cntx As I18NSessionContext = Me.getInnovator().getI18NSessionContext()
Dim date_value As String = cntx.ConvertToNeutral(formatted_date, "date", "short_date")
Dim float_value As String = cntx.ConvertToNeutral(float_num, "float", "")
Dim decimal_value As String = cntx.ConvertToNeutral(decimal_num, "decimal", "")

Return Me.getInnovator().newResult(date_value + " " + float_value + " " + decimal_value)

Notes:

Accepts the following parameters:

sValue - Locale specific string representation of a value that has to be converted.
vtype - Type of the value; only the following types are considered to be locale specific - 'date' | 'float' | 'decimal'
datePtrn - This parameter is taken into consideration only if vtype='date'. In this case the parameter can have one the following values:
short_date,  long_date,  short_date_time,  long_date_time,  short_time,  long_time

In the examples above if the session context fr_FR (French - France) the following neutral values are returned:

<Result>2013-05-02T00:00:00 41.000273576291271 234.12</Result>
