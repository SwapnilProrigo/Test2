AML Actions

delete

Remove an Item(s) from the server.

Syntax:

<Item type="Type" action="delete" [item attributes]>
</Item>
 

Notes:

The edit action requires item attribute be either an existing GUID or a where clause to prevent an ambiguous query.

Examples:

<!-- Delete specified part -->
<Item type="Part" action="delete" where="[part].item_number='ABCD-1234'">
</Item>

<!-- Delete existing parts that meet criteria -->
<Item type="Part" action="delete" where="[part].state='Preliminary'">
</Item>
