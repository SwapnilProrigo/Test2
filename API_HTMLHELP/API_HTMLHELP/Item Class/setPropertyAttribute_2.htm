Item Class

setPropertyAttribute(String propertyName, String attributeName, String language)

Sets the specified attribute of the property with the specified value using language.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "A*", "fr");
itm.setPropertyAttribute("name", "condition", "like", "fr");
return itm.apply();
 

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "A*")
itm.setPropertyAttribute("name", "condition", "like", "fr")
Return itm.apply()
 

Notes:

If property tag with the specified name and language doesn't exist in this.node, the property tag is created after which the attribute is set on it. Note, that if the property tag is created by the method, the property value (i.e. tag's InnerText) is empty.
