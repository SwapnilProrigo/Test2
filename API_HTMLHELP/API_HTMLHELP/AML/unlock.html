AML Actions

unlock

Unlock an Item - remove locked_by_id property setting.

Syntax:

<Item type="Type" action= "unlock" [item attributes]>
</Item>
 

Notes:

The unlock action requires item attribute id be set to an existing 32 character GUID (guaranteed unique identifier) or GUID list.

Examples:

<!-- Lock specified part -->
<Item type="Part" action="unlock" id="ABCD1234EFGH5678IJKLM90123NOPQR">
</Item>
