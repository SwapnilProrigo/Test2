Innovator Class

newItem(String itemType, String action)

Creates a new AML request with the specified itemType and action.

C#:

Innovator inn = this.getInnovator();
Item itm = inn.newItem("Customer", "get");
return itm.apply();

VB:

Dim inn As Innovator = Me.getInnovator()
Dim itm As Item = inn.newItem("Customer", "get")
Return itm.apply()

Notes:

Item request is established with supplied ItemType and action.
