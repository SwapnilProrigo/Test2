Item Class

getResult()

Returns 'InnerText' property of Result item

JavaScript:

//Calls server method below which returns Result Item
var inn = this.getInnovator();
var itemResult = inn.applyMethod("_ItemCount", "");
var result = itemResult.getResult();
alert(result);

C#

//_ItemCount server method
Innovator inn = this.getInnovator();
Item itemcount = inn.applySQL("select count(*) as count from innovator.part");
string count = itemcount.getProperty("count");
return inn.newResult(count);

VB:

'_ItemCount server method
Dim inn As Innovator = Me.getInnovator()
Dim itemcount As Item = inn.applySQL("select count(*) as count from innovator.part")
Dim count As String = itemcount.getProperty("count")
Return inn.newResult(count)

Notes:

If this.dom is null or node with Result was not found, the method returns an empty string.

In the example above a JavaScript client Method uses Innovator.applyMethod to call a server Method that gets the count of all Parts in the database.  The server Methods return a new Result item which is then parsed by the JavaScript client Method to get the count value.
