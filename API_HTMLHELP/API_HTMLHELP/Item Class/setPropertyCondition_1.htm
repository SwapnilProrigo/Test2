Item Class

setPropertyCondition(String propertyName, String condition)

Sets the specified attribute of the property with the specified condition.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "A*");
string state = itm.setPropertyCondition("name", "like");
return itm.apply();

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "A*")
Dim state As String = itm.setPropertyCondition("name", "like")
Return itm.apply();

Notes:

If property tag with the specified name doesn't exist in this.node, the property tag is created after which the attribute 'condition' is set on it. Note, that if the property tag is created by the method, the property value (i.e. tag's InnerText) is empty. See "Innovator Programmer's Guide" for more details on condition syntax.
