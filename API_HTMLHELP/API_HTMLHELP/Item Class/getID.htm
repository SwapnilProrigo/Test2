Item Class

getID()

Returns ID of the Item node. According to AML standard ID could be set on <Item> either as the attribute with name 'id' or as a sub-tag <id> (i.e. item property) or both.

C#:

Item itm = this.newItem("Customer", "add");
itm.setProperty("name", "ACME");
itm.setProperty("contact_name", "Andrew Harris");
itm = itm.apply();
string id = itm.getID();

VB:

Dim itm As Item = Me.newItem("Customer", "add")
itm.setProperty("name", "ACME")
itm.setProperty("contact_name", "Andrew Harris")
itm = itm.apply()
Dim id As String = itm.getID()

Notes:

ID of the item or empty string if ID was not found.
