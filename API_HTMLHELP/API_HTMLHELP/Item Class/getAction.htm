Item Class

getAction()

Returns the 'action' attribute from the Item node.

C#

if (this.getAction()=="get") {
 this.setAction("edit");
}
return this;

VB:

if (this.getAction()=="get") {
 this.setAction("edit");
}
return this;

Notes:

The method returns the value of the 'action' attribute if the attribute exists, empty string otherwise.

The examples above check to see if the current item action is "get" and change it to "edit" if the condition is true.
