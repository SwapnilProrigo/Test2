Item Class

setPropertyItem(String propName, Item argItem)

Set an item property to argItem

C#

Item identity = this.newItem("Identity", "get");
identity.setProperty("name", "Component Engineering");
Item result_iden = identity.apply();

Item part = this.newItem("Part" , "add");
part.setProperty("name", "Cable Harness");
part.setProperty("item_number", "CH-1022");
part.setPropertyItem("owned_by_id", result_iden);
return part.apply();

VB:

Dim identity As Item = Me.newItem("Identity", "get")
identity.setProperty("name", "Component Engineering")
Dim result_iden as Item = identity.apply()

Dim part As Item = Me.newItem("Part", "add")
part.setProperty("name", "Cable Harness")
part.setProperty("item_number", "CH-1022")
part.setPropertyItem("owned_by_id", result_iden)
Return part.apply()

Notes:

If for some reason an item can't be returned (e.g. 'this.dom' doesn't have a property with the specified name or the property contains just id but type of the item-property is unknown), the method returns null.

This method sets the item property to a created item.  To create an item and assign property you can also use the createPropertyItem  Method.
