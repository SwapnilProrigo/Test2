Innovator Class

getI18NSessionContext()

Returns current internationalization context.

C#:

Innovator inn = this.getInnovator();
I18NSessionContext i = inn.getI18NSessionContext();
return inn.newResult(i.GetLanguageCode())();
 

VB:

Dim inn As Innovator = Me.getInnovator()
Dim i As I18NSessionContext = inn.getI18NSessionContext()
Return inn.newResult(i.GetLanguageCode())
 

Notes:

See I18NSessionContext  class for methods to parse returned context.
