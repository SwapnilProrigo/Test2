Item Class

getRelatedItemID()

Gets related item ID of the relationship.

C#:

Item vendor = this.newItem("Vendor", "get");
vendor.setProperty("name", "Allied Industries");
vendor = vendor.apply();
vendor.fetchRelationships("Vendor Part");
Item relationships = vendor.getRelationships("Vendor Part");
Item partRel = relationships.getItemByIndex(0);
string vendorPartID = partRel.getRelatedItemID();
return this.getInnovator().newResult(vendorPartID);

VB:

Dim vendor As Item = Me.newItem("Vendor", "get")
vendor.setProperty("name", "Allied Industries")
vendor = vendor.apply()
vendor.fetchRelationships("Vendor Part")
Dim relationships As Item = vendor.getRelationships("Vendor Part")
Dim partRel As Item = relationships.getItemByIndex(0)
Dim vendorPartID As String = partRel.getRelatedItemID()
Return Me.getInnovator().newResult(vendorPartID)

Notes:

If related item not found, returns empty string.

The examples above return the first Manufactured Part ID associated with a Vendor.
