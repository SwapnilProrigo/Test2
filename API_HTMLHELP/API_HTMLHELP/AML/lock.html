AML Actions

lock

Lock an Item - set locked_by_id property to current user.

Syntax:

<Item type="Type" action="lock"[item attributes]>
</Item>
 

Notes:

The lock action requires item attribute id be set to an existing 32 character GUID (guaranteed unique identifier) or GUID list.

Examples:

<!-- Lock specified part -->
<Item type="Part" action="lock" id="ABCD1234EFGH5678IJKLM90123NOPQR">
</Item>
