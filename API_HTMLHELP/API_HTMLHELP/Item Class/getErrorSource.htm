Item Class

getErrorSource()

Returns the content of the <faultactor> element of SOAP Fault element.

C#:

string faultString="";
Item part = this.newItem("Program", "get");
part = part.apply();

if (part.isError()) {
 faultString = part.getErrorSource();
 }

return this.getInnovator().newResult(faultString);

VB:

Dim faultStringAs String = ""
Dim part As Item = Me.newItem("Program", "get")
part = part.apply()

If part.isError() Then
 faultString= part.getErrorSource()
End If

Return Me.getInnovator().newResult(faultString)

Notes:

If the instance is not an error item, an empty string is returned.
