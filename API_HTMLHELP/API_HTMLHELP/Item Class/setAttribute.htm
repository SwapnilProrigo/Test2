Item Class

setAttribute(String attributeName, String attributeValue)

Set the item attribute specified.

C#:

Item part = this.newItem("Part");
part.setAction("get");
part.setAttribute("select", "item_number,name");
return part = part.apply();

VB:

Dim part As Item = Me.newItem("Part")
part.setAction("get")
part.setAttribute("select", "item_number,name")
Return
 

Notes:

Set the item attribute to specified value.
