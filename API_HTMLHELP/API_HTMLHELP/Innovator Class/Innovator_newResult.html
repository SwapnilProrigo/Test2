Innovator Class

newResult(String text)

Returns an Item where the string argument becomes the body of the enclosed <Result> tag.

C#:

Innovator inn = this.getInnovator();
Item itm = inn.newItem("User", "get");
itm.setProperty("login_name", "admin");
itm = itm.apply();
return inn.newResult(itm.getProperty("last_name"));

VB:

Dim inn As Innovator = Me.getInnovator()
Dim itm As Item = inn.newItem("User", "get")
itm.setProperty("login_name", "admin")
itm = itm.apply()
Return inn.newResult(itm.getProperty("last_name"))

Notes:

After the insertion the method performs a verification that resulting AML is well-formed XML.

If passed text contains any XML escape symbols it is the user's responsibility to put the text inside a 'CDATA' block.
