Item Class

getPropertyItem(string propertyName)

Returns the item node of a property of datatype Item..

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME");
itm = itm.apply();
Item creator = itm.getPropertyItem("created_by_id");
string cid = creator.getID();
return this.getInnovator().newResult(cid);
 

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME")
itm = itm.apply()
Dim creator As Item = itm.getPropertyItem("created_by_id")
Dim cid As String = creator.getID()
Return Me.getInnovator().newResult(cid)

Notes:

Returns found item associated with the property.
If the Item node has not been established for the property in the dom then the method returns a "simple" item in the form of <Item><id>...</id></Item>
