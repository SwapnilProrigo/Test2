Item Class

setPropertyAttribute(String propertyName, String attributeName, String value)

Sets the specified attribute of the property with the specified value.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "A*");
itm.setPropertyAttribute("name", "condition", "like");
return itm.apply();

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "A*")
itm.setPropertyAttribute("name", "condition", "like")
Return itm.apply();

Notes:

If either property with the specified name or the specified attribute on the property doesn't exist, returns null ( in JavaScript, returns undefined.)
To set the value of a property to NULL use the is_null attributeName set to "1" - e.g.  itm.setPropertyAttribute("name", "is_null", "1")
