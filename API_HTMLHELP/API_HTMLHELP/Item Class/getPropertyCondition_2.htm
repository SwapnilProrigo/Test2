Item Class

getPropertyCondition(string propertyName, string lang)
 

Gets the 'condition' attribute of the property with the specified name and language.

C#:

string condition = itm.getPropertyCondition("name", "fr");

VB:

Dim condition As String = itm.getPropertyCondition("name", "fr")

Notes:

If either property with the specified name or attribute 'condition' on the property doesn't exist, returns null ( in JavaScript, returns undefined.)
