Item Class

fetchLockStatus()

Fetches the locked status of the item from the database using the item's ID.

C#

Item customer = this.newItem("Customer", "get");
customer.setID("F045495F51854DC38200B73EFC116130");
int lockStatus = customer.fetchLockStatus();

VB:

Dim customer As Item = Me.newItem("Customer", "get")
customer.setID("F045495F51854DC38200B73EFC116130")
Dim lockStatus As Integer = customer.fetchLockStatus()

Notes:

The method sends a request to the server in order to fetch the latest lock status of the item on server and updates the item's property locked_by_id. An apply is not required.

Possible integer returns:

-1 failed to fetch item's lock status from server
0 not locked
1 locked by user
2 locked by other
