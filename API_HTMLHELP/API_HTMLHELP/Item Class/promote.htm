Innovator Class

promote(String state, String comments)
 

Promote item to the specified state.

C#:

Item vendor = this.newItem("Vendor", "get");
vendor.setProperty("name", "Allied Industries");
vendor = vendor.apply();

if (vendor.getProperty("state")=="Preliminary") {
 vendor.promote("Approved", "auto approved");
}
return vendor;

VB:

Dim vendor As Item = Me.newItem("Vendor", "get")
vendor.setProperty("name", "Allied Industries")
vendor = vendor.apply()

If vendor.getProperty("state") = "Preliminary" Then
 vendor.promote("Approved", "auto approved")
End If
Return vendor

Notes:

If ItemType of the item is configured to log "history" records when items of the ItemType are created\promoted\etc., (i.e. if the ItemType's property "save_history" is set to "1") then the comments will be used as a comment for the "history" record about the promote event.

Promote state specifiec must be next available promotion state in lifecycle map and user must be a member of the transition identity defined on the lifecycle map.

An apply() is NOT required after a promote.
