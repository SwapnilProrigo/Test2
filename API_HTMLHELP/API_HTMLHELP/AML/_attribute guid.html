AML Attributes

GUID

Every item in Aras Innovator is assigned a guaranteed unique identifier which consists of 32 alphanumeric characters.

Syntax:

id="B7DE10503F3A4C08BA056628921A440E"
 

Notes:

The GUID is assigned the to the "id" property of every Item which is designated as the primary key column.

Examples:

<!-- Purge specified part version-->
<Item type="Part" action="purge" id="B7DE10503F3A4C08BA056628921A440E">
</Item>
