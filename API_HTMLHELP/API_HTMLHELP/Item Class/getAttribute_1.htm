Item Class

getAttribute(String attribute)

Returns value of the attribute with the specified name on the item's node.

C#:

if (this.getAttribute("select")==null) {
 this.setAttribute("select", "id,item_number");
}
return this;
 

VB:

If Me.getAttribute("select") Is Nothing Then
 Me.setAttribute("select", "id,item_number")
End If
Return Me

Notes:

Returns attribute value or null if the attribute doesn't exist.

The examples above check to see if the current item has a select attribute.  If not, then one is set.
