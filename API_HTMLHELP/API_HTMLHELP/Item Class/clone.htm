Item Class

clone(bool cloneRelationships)

Make a duplicate of an item.  if cloneRelationships flag is true then any currently retrieved relationships are also cloned.

C#

Item itm = this.newItem("Vendor" , "get");
itm.setProperty("name", "Allied Industries");
itm.createRelationship("Vendor Part", "get");
itm = itm.apply();

Item i_copy = itm.clone(true);
i_copy.setProperty("name", "Copy of Allied Industries");
return i_copy.apply();

VB:

Dim itm As Item = Me.newItem("Vendor", "get")
itm.setProperty("name", "Allied Industries")
itm.createRelationship("Vendor Part", "get")
itm = itm.apply()

Dim i_copy As Item = itm.clone(True)
i_copy.setProperty("name", "Copy of Allied Industries")
Return i_copy.apply()

Notes:

The method creates a new instance of item that must be applied to be permanently saved.

To clone relationships the original item must be contain the relationship data to be cloned.  In the examples above, the Vendor Part relationships are retrieved along with the parent Vendor item.  When the original Vendor is cloned the Vendor Part relationships declared in the original are also cloned.

Cloned item as well as cloned relationships have new IDs and action attributes set to 'add' which must be applied to the server.
