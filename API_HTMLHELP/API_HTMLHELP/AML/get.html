AML Actions

get

Retrieve items that meet specified criteria.

Syntax:

<Item type = "Type" action= "get" [item attributes]>
 <property [condition="option"]>value</property>
 ...
 <Relationships>
  <Item type="Type" action="get" [item attributes]>
   <property [condition="option"]>value</property>
   ...
     <related_id>
      <Item type="Type" action="get" [item attributes]>
       <property [condition="option"]>value</property>
        ...
      </Item>
     <related_id>
  </Item>
 </Relationships>
</Item>
 

Notes:

Add creates a new Item on the server of the specified type.  Relationships to related Items can also be defined using the <Relationships> tag.

Examples:

<!-- Get part with item number and name specified -->
<Item type="Part" action="get">
 <item_number>ABCD-1234</item_number>
 <name>Transistor</name>
</Item>

<!-- Get parts with name using wild card -->
<Item type="Part" action="get">
 <name condition="like">Trans*</name>
</Item>

<!-- Get parts where related item and quantity is specified -->
<Item type="Part"  action="get">
  <Relationships>
    <Item type="Part BOM" action="get">
      <quantity>1</quantity>
      <related_id>
        <Item type="Part" action="get">
          <item_number>1121-9011</item_number>
        </Item>
      </related_id>
    </Item>
  </Relationships>
</Item>
