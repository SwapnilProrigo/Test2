Item Class

createRelationship(String, String)

Creates new <Item> node with specified 'type' and 'action' under <Relationships> node..

C#:

Item parentItem = this.newItem("Part", "get");
Item relationshipItem = parentItem.createRelationship("Part Document", "get");
Item relatedItem = this.newItem("Document", "get");
relatedItem.setProperty("item_number", "DOC-0001");
relationshipItem.setRelatedItem(relatedItem);
return parentItem.apply();

VB:

Dim parentItem As Item = Me.newItem("Part", "get")
Dim relationshipItem As Item = parentItem.createRelationship("Part Document", "get")
Dim relatedItem As Item = Me.newItem("Document", "get")
relatedItem.setProperty("item_number", "DOC-0001")
relationshipItem.setRelatedItem(relatedItem)
Return parentItem.apply()

Notes:

The method inserts passed relationship Item under <Relationships> node.  Relationship Item node is created and placed under parent node.

Examples above creates the following AML Request:

<Item type="Part" action="get">
<Relationships>
<Item type="Part Document" action="get">
<related_id>
<Item type = "Document" action="get">
<item_number>DOC-0001"</item_number>
</Item>
<related_id>
</Item>
</Relationships>
</Item>
