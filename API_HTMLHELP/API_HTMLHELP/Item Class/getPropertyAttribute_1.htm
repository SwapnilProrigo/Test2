Item Class

getPropertyAttribute(string propertyName, string attributeName)
 

Gets the specified attribute of the property with the specified name.

C#:

Item itm = this.newItem("Customer", "get");
itm.setProperty("name", "ACME");
itm = itm.apply();
string state = itm.getPropertyAttribute("current_state", "keyed_name");
return this.getInnovator().newError(state);

VB:

Dim itm As Item = Me.newItem("Customer", "get")
itm.setProperty("name", "ACME")
itm = itm.apply()
Dim state As String = itm.getPropertyAttribute("current_state", "keyed_name")
Return Me.getInnovator().newError(state)

Notes:

If either property with the specified name or the specified attribute on the property doesn't exist, returns null ( in JavaScript, returns undefined.)
