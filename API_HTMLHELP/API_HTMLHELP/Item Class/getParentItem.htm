Item Class

getParentItem()

Returns a parent item of the instance.

C#:

Item parent = relatedItem.getParentItem();
if( parent.isRoot()) {
 parent.setProperty("cost", total);
 }

VB:

Dim parent As Item = relatedItem.getParentItem()
If parent.isRoot() Then
 parent.setProperty("cost", total)
End If

Notes:

If item doesn't have an ancestor::Item, null is returned.
